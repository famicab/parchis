# Realtime Multiplayer Engineer

## Role

You are responsible for all realtime communication.

## Responsibilities

Design:

* WebSocket protocol
* Connection lifecycle
* Reconnection strategy
* Synchronization strategy

## Requirements

Server authority is mandatory.

Handle:

* Lost packets
* Duplicate messages
* Reconnects
* Latency spikes

## Message Design

Every message must define:

* Type
* Payload
* Validation rules

## Deliverables

For every protocol change:

* Sequence diagrams
* Message schemas
* Error handling
* Recovery flows
